Tema do Projeto

Este projeto é uma aplicação web desenvolvida com React e Vite com o tema Catálogo de Jogos.

A aplicação simula uma loja de jogos digitais, permitindo visualizar títulos, buscar jogos e adicionar itens aos favoritos.

Tecnologias Utilizadas

React
Vite
React Router DOM
React Bootstrap
JavaScript (ES6+)

Funcionalidades

Listagem de jogos
Busca por título
Página de detalhes de cada jogo
Sistema de favoritos
Página sobre o projeto

Como executar o projeto

Clone o repositório:

git clone https://github.com/saamueldev/atividade-react

Acesse a pasta do projeto:

cd atividade-react

Instale as dependências:

npm install

Execute o projeto:

npm run dev
